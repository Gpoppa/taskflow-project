# Taskflow Project
TaskFlow Gym es una aplicación de gestión de entrenamientos diseñada para organizar rutinas de calistenia y gimnasio de forma sencilla e intuitiva. Permite crear, completar y eliminar ejercicios, así como clasificarlos por tipo (Push, Pull, Legs). Guarda el progreso en el navegador mediante LocalStorage y muestra estadísticas básicas para seguir la evolución de los entrenamientos. Además, cuenta con un diseño responsive adaptado tanto a móvil como a escritorio
